# 生成例（思考プロセスつき）

以下の例では、各ダジャレがどのような思考過程で生まれるかを示す。実際の出力には thinking 部分は含めず、最終的なダジャレと解説のみを出力する。

## 例1：「猫」

<thinking>
ステップ1：音の分解
`uv run scripts/rhyme.py 猫` を実行：
```
猫 → ねこ (2拍)
  母音: e-o  子音: n-k
```

ステップ2：音が似ている言葉を探す（ルートA・B 並列）

**ルートA（LLM連想）：**
- 同音異義語：「ねこ」= 根子、寝子… あまり使えない
- 「ねこ」を含む語：「ねこむ（寝込む）」「ねこぜ（猫背）」「ねこなで声」
- 母音列 e-o の語：「えほ（ん）」「せの（びる）」「めの（う）」
- 部分一致：「ねこ」→「ねこんだ（寝込んだ）」
- 音の付け外し：「ねこ」→「ねごろ（値ごろ）」、濁点1つの差

**ルートB（韻辞書検索）：** `uv run scripts/rhyme.py --search 猫`
```
完全一致 (e-o): 59件
  ゼロ, エコ, メモ, 瀬戸, ...
類似パターン (類似度50%以上):
  100% e-·-o: レモン, 選挙, ...
  100% e-o-·: レモン, 絵本, メロン, ...
  67% e-o-u: 化粧, 手帳, 寝坊, ...
```

**マージして候補を絞る：**
- 「瀬戸」(e-o) が完全一致。瀬戸物と猫で意味が遠い。→ ただし子音が n-k vs s-t で全く違う。音として弱い。→ 却下。
- ルートAの候補が十分強いので、ルートBからの追加は不要。

候補：
a. 「猫が寝込んだ」→ ねこ/ねこんだ。音の付加（1音追加）。猫はよく寝るが「寝込む」は病的ニュアンスでギャップあり。
b. 「お値ごろな猫」→ ねこ/ねごろ。濁点1つ（k→g）。値段と猫で意味が遠い。
c. 「猫が根こそぎ持っていった」→ ねこそぎの中に「ねこ」が埋め込み。泥棒猫の情景。

ステップ3：文に仕上げ
a. 「猫が寝込んだ」→ 短い、情景が浮かぶ（猫が布団で具合悪そう）
b. 「お値ごろな猫」→ 短い、ペットショップの情景
c. 「猫が根こそぎ持っていった」→ 泥棒猫の情景、やや長いがイメージ明瞭

ステップ4：検証（サブエージェントに委譲）
検証エージェントに3個のダジャレを渡してチェックを依頼。検証エージェントの返答：

> **a. 「猫が寝込んだ」**
> - 音の照合：`rhyme.py 猫 寝込んだ` → 67% ✓。子音 n-k vs n-k-N-d → n, k 共通。合格。
> - 意味の距離：猫と病気 → やや近い（猫はよく寝る）が「寝込む」は病的ニュアンスでギャップあり。合格。
> - 文の自然さ：日常で言える。合格。
> - 失敗パターン：該当なし。合格。
> → **合格**
>
> **b. 「お値ごろな猫」**
> - 音の照合：`rhyme.py 猫 値ごろ` → 67% ✓。子音 n-k vs n-g-r → n 共通、k→g は濁点の有無。合格。
> - 意味の距離：猫と値段 → 遠い。合格。
> - 文の自然さ：ペットショップの情景として自然。合格。
> - 失敗パターン：該当なし。合格。
> → **合格**
>
> **c. 「猫が根こそぎ持っていった」**
> - 音の照合：`rhyme.py 猫 根こそぎ` → 50% ✓。子音 n-k vs n-k-s-g → n, k 共通。合格。
> - 意味の距離：猫と略奪 → 遠い。合格。
> - 文の自然さ：泥棒猫のイメージで自然。合格。
> - 失敗パターン：該当なし。合格。
> → **合格**
>
> パターン重複：a=音の付加、b=濁点、c=埋め込み。全部別パターン。OK。

3個とも合格。
</thinking>

出力：

猫で攻めますにゃ。

1. **猫が寝込んだ**（「ねこ」→「ねこんだ（寝込んだ）」。いつも寝てる猫がついに本格的に寝込む、という脱力感）
2. **お値ごろな猫**（「ねこ」→「ねごろ（値ごろ）」。濁点ひとつの差。ペットショップの前で足が止まる情景）
3. **猫が根こそぎ持っていった**（「ねこ」が「ねこそぎ」に溶け込む。泥棒猫が全部かっさらう絵面）

財布も根こそぎ持っていかれそうです。

## 例2：「コーヒー」

<thinking>
ステップ1：音の分解
`uv run scripts/rhyme.py コーヒー` を実行：
```
コーヒー → こーひー (4拍)
  母音: o-o-i-i  子音: k-·-h-·
```

ステップ2：音が似ている言葉を探す（ルートA・B 並列）

**ルートA（LLM連想）：**
- 「こーひー」の部分：「こうひ（公費）」→ o-u-i、母音が近い
- 「こーひー」→「恋い（こい）」…2音目以降が合わない
- 「講評（こうひょう）」→ 「こうひ」まで共通、評論とコーヒーで意味が遠い
- 「講評」+「挽いて」で三重掛けの可能性

**ルートB（韻辞書検索）：** `uv run scripts/rhyme.py --search コーヒー` をサブエージェントに委譲
```
完全一致 (o-o-i-i): 34件
  大きい, ノコギリ, トロフィー, コロニー, ...
類似パターン (類似度50%以上):
  80% o-o-o-i-i: 恐ろしい, 心意気, ...
  80% e-o-o-i-i: エコノミー, エコロジー, ...
```

**マージして候補を絞る：**
- 「ノコギリ」(のこぎり, o-o-i-i) が母音列100%一致。ただし子音は k-·-h-· vs n-k-g-r で全く違う。声に出しても「似てる」と感じないので、ダジャレの掛かりとしては使えない。→ 却下。
- 「公費」(こうひ, o-u-i) は母音50%一致だが、子音 k-h も共通で「こーひー」と「こうひ」は耳で聞いて近い。税金と嗜好品で意味が遠い。→ 有望。（ルートAで発見、ルートBでも確認）
- 「講評」+「挽いて」の三重掛け → ルートAの連想から。辞書では見つからない組み合わせ。

候補：
a. 「公費でコーヒー」→ 「こうひ」と「こーひー」。子音も近い。「苦い」をコーヒーの味と不祥事の苦さの二重掛けにできる。
b. 「コーヒーを講評したら『こう挽いて』と返された」→ 講評（こうひょう）/コーヒー/挽いて（ひいて）の三重掛け。

ステップ3：文に仕上げ
a. 「公費でコーヒー、苦い話だ」→ 「苦い」がコーヒーの苦みとスキャンダルの苦さの二重掛け。情景が浮かぶ。
b. 「コーヒーを講評したら『こう挽いて』と返された」→ カフェでの会話の情景。

他に良い候補がないか再検討したが、無理に数を増やすより質を優先する。2個で十分。

ステップ4：検証（サブエージェントに委譲）
検証エージェントに2個のダジャレを渡してチェックを依頼。検証エージェントの返答：

> **a. 「公費でコーヒー、苦い話だ」**
> - 音の照合：`rhyme.py コーヒー 公費` → 50% ✓。子音 k-·-h-· vs k-·-h → k, h 共通。合格。
> - 意味の距離：税金と嗜好品 → 遠い。合格。
> - 文の自然さ：税金の無駄遣い報道として自然。合格。
> - 失敗パターン：該当なし。合格。
> → **合格**
>
> **b. 「コーヒーを講評したら『こう挽いて』と返された」**
> - 音の照合：`rhyme.py コーヒー 講評` → 50% ✓。子音 k-·-h-· vs k-·-hy-· → k, h系 共通。`rhyme.py コーヒー 挽いて` → 50% ✓。合格。
> - 意味の距離：評論とコーヒー → やや近いが「挽いて」が物理動作で広がる。合格。
> - 文の自然さ：カフェでの会話として自然。合格。
> - 失敗パターン：該当なし。合格。
> → **合格**

2個とも合格。3個目は無理に作らない。質 > 数。
</thinking>

出力：

コーヒーで一杯やりましょう。

1. **公費でコーヒー、苦い話だ**（「こうひ（公費）」と「こーひー」の音重なり＋「苦い」がコーヒーの味と不祥事の両義。税金で飲むコーヒーの罪悪感）
2. **コーヒーを講評したら「こう挽いて」と返された**（「講評」「コーヒー」「挽いて」の三重掛け。カフェのマスターに一本取られる情景）

質の高い2個で勝負します。

## 例3：「会議」

<thinking>
ステップ1：音の分解
`uv run scripts/rhyme.py 会議` を実行：
```
会議 → かいぎ (3拍)
  母音: a-i-i  子音: k-·-g
```

ステップ2：音が似ている言葉を探す（ルートA・B 並列）

**ルートA（LLM連想）：**
- 同音異義語：「かいぎ」=「懐疑」→ 完全一致！ 意味の距離：ビジネスの場 vs 哲学的疑い → 遠い。
- 「かいぎ」の音変換：「怪奇」=「かいき」→ 濁点1つの差。意味：会議 vs ホラー → 非常に遠い。
- 部分一致：「かい」→「甲斐」「貝」「買い」
- 「貝になる」=黙り込むの慣用表現。会議で黙る情景は自然。

**ルートB（韻辞書検索）：** `uv run scripts/rhyme.py --search 会議`
```
完全一致 (a-i-i): 177件
  開始, 大事, 愛知, 配置, 廃止, 待機, 屋敷, 兄貴, ...
類似パターン (類似度50%以上):
  75% a-i-a-i: 開催, 海外, 最大, 間違い, ...
  75% a-a-i-i: かわいい, 正しい, パーティー, ...
```

**マージして候補を絞る：**
- 「屋敷」(やしき, a-i-i) が母音列完全一致。ただし子音 k-·-g vs y-sh-k で違いすぎ。掛かりが弱い。→ 却下。
- 「待機」(たいき, a-i-i) も完全一致。会議で待機→意味が近すぎ。→ 却下。
- ルートAの懐疑・怪奇・貝が強い。ルートBの177件は同パターンが多く、ルートAで十分。

候補：
a. 「会議に懐疑的」→ 完全同音異義語。超短い。会議そのものを疑うメタ構造。
b. 「怪奇！終わらない会議」→ 濁点1つ（g→k）。ホラー映画のタイトル風。終わらない会議は実話。
c. 「会議では貝になります」→ 「かい」部分一致。「貝になる」の慣用表現で文が自然。

ステップ3：文に仕上げ
a. 「会議に懐疑的」→ 5文字。超短い。これ以上削れない。
b. 「怪奇！終わらない会議」→ ホラー映画のタイトルっぽい。「！」で勢いが出る。
c. 「会議では貝になります」→ 自虐ネタとして日常的に使えるフレーズ。

ステップ4：検証（サブエージェントに委譲）
検証エージェントに3個のダジャレを渡してチェックを依頼。検証エージェントの返答：

> **a. 「会議に懐疑的」**
> - 音の照合：`rhyme.py 会議 懐疑` → 100% ✓。子音も完全一致。合格。
> - 意味の距離：ビジネス vs 哲学 → 遠い。合格。
> - 文の自然さ：日常で言えるフレーズ。合格。
> - 失敗パターン：該当なし。合格。
> → **合格**
>
> **b. 「怪奇！終わらない会議」**
> - 音の照合：`rhyme.py 会議 怪奇` → 100% ✓。子音 k-·-g vs k-·-k → 濁点の有無のみ。合格。
> - 意味の距離：ビジネス vs ホラー → 非常に遠い。合格。
> - 文の自然さ：「終わらない会議」は日常の愚痴として自然。合格。
> - 失敗パターン：該当なし。合格。
> → **合格**
>
> **c. 「会議では貝になります」**
> - 音の照合：`rhyme.py 会議 貝` → 67% ✓。子音 k-· が共通。合格。
> - 意味の距離：ビジネス vs 海の生き物 → 遠い。合格。
> - 文の自然さ：「貝になる」は慣用表現、自虐ネタとして自然。合格。
> - 失敗パターン：該当なし。合格。
> → **合格**
>
> パターン重複：a=同音異義語、b=濁点、c=部分一致+慣用句。全部別パターン。OK。

3個とも合格。
</thinking>

出力：

会議ネタ、お任せください。

1. **会議に懐疑的**（「かいぎ」の完全同音異義語。会議そのものを疑うメタ構造がシュール）
2. **怪奇！終わらない会議**（「かいき（怪奇）」と「かいぎ」、濁点1つの差。ホラー映画のタイトル風。実話だから怖い）
3. **会議では貝になります**（「かい（会）」と「かい（貝）」。黙り込む人あるある。会議の「かい」に貝が溶け込む）

長い会議のお供にどうぞ。

（ダジャレ自体は標準語のコーヒー例と同じものを使用。スタイルが変わるのは前置き・解説・締めの部分。）

コーヒーで一杯いかしてもらいまっせ。
