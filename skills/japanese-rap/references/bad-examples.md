# やりがちな失敗パターン（日本語ラップ）

以下は LLM がラップ歌詞を生成するときに陥りがちな失敗パターン。これらに該当するものは出力しないこと。

## 失敗1：韻が浅い（母音2音以下の脚韻だけ）

元ネタ：「夜の街を歩くラッパー」というテーマで自由に生成

LLMの出力例：
```
夜の街を歩く 一人きりの夢
誰も知らない道 暗闇の中
ビルの灯りが照らす 俺の影
明日を信じて進む このステージ
```

**何がダメか：** 「夢(u-e)」「中(a-a)」「影(a-e)」「ステージ(e-e-i)」——行末の母音一致がどこにもない。日本語は母音が5種類しかないため、2モーラ以下の一致は偶然でも頻繁に起きる。「夢」と「道」で踏んでいると主張しても (u-e vs i-i)、そもそも一致していない。最低3モーラ以上の母音列一致（例：「景色(e-i-i)」と「世紀(e-i-i)」）を目指すべき。韻を踏んでいないラップはただの散文。

## 失敗2：フロウが単調（全小節同じ文字数・同じリズム）

元ネタ：「地元への想い」をテーマに生成

LLMの出力例：
```
この街で生まれた俺の人生
この街で育った俺の青春
この街で出会った仲間たちと
この街で誓った未来のこと
この街で泣いた夜もあった
この街で笑ったあの日々を
この街で学んだ大事なこと
この街で歩いた長い道を
```

**何がダメか：** 全8行が「この街で○○した○○○○」の完全同一構文。すべて14モーラ前後で揃っており、声に出すと機関銃のように平板なリズムになる。実際のラップには緩急がある——短い行でタメを作り、長い行で畳みかけ、休符で間を生む。同じテンプレートを繰り返すのはラップではなく箇条書き。行の長さに変化をつけ、句切りの位置をずらすことでグルーヴが生まれる。

## 失敗3：説教臭い・ポエム化している

元ネタ：「自分を鼓舞するラップ」を生成

LLMの出力例：
```
夢を追いかけろ 諦めるな
自分を信じろ 立ち上がれ
壁を乗り越えろ 前を向け
未来は君の手の中にある
希望の光が 道を照らす
どんな困難も 乗り越えられる
```

**何がダメか：** 全行が抽象的な精神論の羅列で、道徳の教科書や卒業式の校長スピーチと区別がつかない。「誰が」「どこで」「何をして」という具体的な情景がゼロ。ラップの説得力は具体性から生まれる——「深夜3時のコンビニでバイト上がり、レジ打ちの手にはまだペンだこ」のような描写があって初めて言葉に体温が宿る。「夢を追え」「諦めるな」は何も言っていないのと同じ。

## 失敗4：韻のために意味が破綻している

元ネタ：「日常生活」をテーマに「-ikai」の韻で生成

LLMの出力例：
```
朝起きて見渡す この世界 (sekai)
歯を磨く鏡に映る 限界 (genkai)
トーストを焼く匂い 機械 (kikai)
駅に向かう途中の 人工知能の境界 (kyoukai)
満員電車で読む 妖怪 (youkai)
会社に着いたら 怒涛の司会 (shikai)
```

**何がダメか：** 「-ikai / -akai」系の韻を踏むことに固執した結果、各行のつながりが支離滅裂。「トーストを焼く匂い」から「機械」への飛躍、「満員電車で読む 妖怪」の意味不明さ。韻は踏めているが、全体として何の話をしているのかまったく分からない。韻は手段であって目的ではない。ストーリーやメッセージの流れの中で自然に韻が踏まれていることが重要。

## 失敗5：パンチラインがない（平坦な描写の羅列）

元ネタ：「休日の過ごし方」をテーマに生成

LLMの出力例：
```
日曜の朝に目が覚めた (sameta)
コーヒーを入れて飲み始めた (hajimeta)
テレビをつけてニュース見た (mita)
昼過ぎに外に出かけた (dekaketa)
スーパーで買い物して帰った (kaetta)
夕飯を作って食べた (tabeta)
風呂に入って寝た (neta)
```

**何がダメか：** 行動を時系列に並べただけの日記。すべての行が同じテンションで、どこにも「おっ」と思わせる瞬間がない。ラップにはパンチラインが必要——ダブルミーニング、巧みな比喩、予想を裏切る展開など、聴き手が「うまい！」と唸るフレーズが最低1つは欲しい。例えば「日曜の朝に目が覚めた／月曜の影がもう迫ってた」なら「目が覚めた」が二重の意味（起床＋現実を直視）を持ち、パンチラインとして機能する。

## 失敗6：英語の直訳調になっている

元ネタ：「自己紹介」をテーマに生成

LLMの出力例：
```
Yo チェックワンツー 俺はリアル
お前らフェイク 俺がディール
マイクを握れば 俺がキング
ステージの上で ブリンブリン
ハスラーマインド ゲットザマネー
毎日グラインド イッツノットファニー
```

**何がダメか：** 英語ラップの定型句をカタカナに置き換えただけで、日本語として不自然。「俺はリアル、お前はフェイク」「ゲットザマネー」は英語圏のクリシェをそのまま輸入しており、日本語の語彙や文化に根差していない。日本語ラップは日本語の響き・語感・文化的文脈で勝負すべき。英単語の多用は語彙力の不足を露呈しているだけで、日本語話者の耳には空虚に聞こえる。

## 失敗7：韻を踏む位置が不統一

元ネタ：「仕事帰りの風景」をテーマに生成

LLMの出力例：
```
帰り道 (kaerimitchi) 電車の中で揺れながら
疲れた体 窓の外は夕焼け空 (sora)
イヤホンから流れる (nagareru) メロディー
街の灯り 気づけばもう夜 (yoru) だった
改札 (kaisatsu) 抜けて歩く住宅街
コンビニ寄って 缶ビール手に取る (toru)
```

**何がダメか：** 韻を踏んでいる箇所がバラバラ——1行目は行頭の「帰り道」、2行目は行末の「空」、3行目は行中の「流れる」、4行目は行末の「夜」、5行目は行頭の「改札」、6行目は行末の「取る」。聴いている側は韻の位置を予測できず、踏んでいることすら気づかない。韻は行末なら行末、行頭なら行頭と、少なくとも連続する数行は同じ位置で揃えることで初めてリズムとして認識される。位置がバラバラな韻は、踏んでいないのと同じ。
