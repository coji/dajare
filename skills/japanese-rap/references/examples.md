# 生成例（思考プロセスつき）

以下の例では、日本語ラップのリリックがどのような思考過程で生まれるかを示す。実際の出力には thinking 部分は含めず、最終的なリリックと韻の解説のみを出力する。

## 例1：「コーヒー」（16小節）

<thinking>
ステップ1：お題の解釈
「コーヒー」にまつわる情景・感情を洗い出す：
- 朝の一杯、目覚めの儀式
- カフェでの孤独な時間
- 苦味と深み（人生の比喩になる）
- 豆を挽く、ドリップする所作
- 眠気覚まし、ブレイクタイム
- 香りに癒される瞬間
- 豆の産地、焙煎のこだわり
- 湯気の向こうに見える景色

ステップ2：コンセプト決定
朝の出勤前にコーヒーを淹れる日常を描く。内省的でありながら、コーヒーを淹れる行為を「自分を整える儀式」として描写する。少しユーモアを混ぜつつ、朝のルーティンが自分のストーリーを始める合図だというメッセージに着地させる。

起承転結の構成：
- 起（1-4小節）：朝、目覚めてコーヒーを淹れる描写
- 承（5-8小節）：コーヒーの苦味と日常の重なり
- 転（9-12小節）：カフェでの内省、孤独と没頭
- 結（13-16小節）：一杯が自分を動かす力になるという着地

ステップ3：キーワードの音分解
`uv run scripts/rhyme.py コーヒー` を実行：
```
コーヒー → こーひー (4拍)
  母音: o-o-i-i  子音: k-·-h-·
```

関連キーワードも分解：

ステップ4：韻ペア探索

**ルートA（LLM連想）で韻候補を列挙：**
- 「コーヒー」(o-o-i-i) → 「トロフィー」(o-o-i-i) 100%一致、「ストーリー」(u-o-o-i-i) 80%一致
- 「香り」(a-o-i) → 「語り」(a-a-i) 67%、「灯り」(a-a-i) 67%
- 「孤独」(o-o-u) → 「没頭」(o-·-o-u) 100%一致
- 「ドリップ」(o-i-·-u) → 「スキップ」(u-i-·-u) 67%
- 「日常」(i-i-o-u) → 「非常」(i-o-u) 75%
- 「苦い」(i-a-i) → 「一杯」(i-i-a-i) 75%
- 「限界」(e-·-a-i) → 「世界」(e-a-i) 100%一致

**ルートB（韻辞書検索）で補強：**
`uv run scripts/rhyme.py --search コーヒー`
```
完全一致 (o-o-i-i): 34件
  大きい, トロフィー, コロニー, ドロシー, ...
類似パターン (類似度50%以上):
  80% o-o-i-o-i: 面白い, 思い込み, ...
  80% u-o-o-i-i: ストーリー, スモーキー
```

**マージして韻ペアを確定：**
- ペアA：「香り」/「語り」 → a-o-i / a-a-i、67%、子音 k-·-r / k-t-r（k, r共通）。起パートの情景描写に使える。
- ペアB：「孤独」/「没頭」 → o-o-u / o-·-o-u、100%。転パートの内省に最適。
- ペアC：「ドリップ」/「スキップ」 → o-i-·-u / u-i-·-u、67%、促音の位置が同じで音感が近い。
- ペアD：「コーヒー」/「トロフィー」 → o-o-i-i / o-o-i-i、100%完全一致。パンチラインに使う。
- ペアE：「コーヒー」/「ストーリー」 → o-o-i-i / u-o-o-i-i、80%。結パートの締めに使う。
- ペアF：「苦い」/「一杯」 → i-a-i / i-i-a-i、75%。コーヒーの味の描写に自然。
- ペアG：「日常」/「非常」 → i-i-o-u / i-o-u、75%。承パートで日常の反転を描ける。
- ペアH：「限界」/「世界」 → e-·-a-i / e-a-i、100%一致。パンチラインに使える。

ステップ5：リリック構築
韻ペアを各小節末に配置し、起承転結に沿って肉付けする。各行12〜18文字を目安にする。

**起（1-4小節）：朝の情景**
1行目末「香り」→ 2行目末「語り」（ペアA）
3行目末「ドリップ」→ 4行目末「スキップ」（ペアC）

**承（5-8小節）：苦味と日常**
5行目末「一杯」→ 6行目末「苦い」（ペアF）
7行目末「日常」→ 8行目末「非常」（ペアG）

**転（9-12小節）：内省と没頭**
9行目末「孤独」→ 10行目末「没頭」（ペアB）
11行目末「世界」→ 12行目末「限界」（ペアH）

**結（13-16小節）：着地**
13行目末「灯り」→ 14行目末「語り」（ペアA変形）
15行目末「コーヒー」→ 16行目末「ストーリー」（ペアE）

ステップ6：検証

**韻の照合：**
- ペアA「香り/語り」：`rhyme.py 香り 語り` → 67% ✓（3モーラ中2モーラ一致、子音 k共通）
- ペアC「ドリップ/スキップ」：`rhyme.py ドリップ スキップ` → 67% ✓（促音位置同一、-i-Q-pu の音感一致）
- ペアF「一杯/苦い」：`rhyme.py 一杯 苦い` → 75% ✓（a-i が共通）
- ペアG「日常/非常」：`rhyme.py 日常 非常` → 75% ✓（i-o-u が共通）
- ペアB「孤独/没頭」：`rhyme.py 孤独 没頭` → 100% ✓（o-o-u 完全一致）
- ペアH「世界/限界」：`rhyme.py 世界 限界` → 100% ✓（e-a-i 完全一致）
- 「灯り/物語り」：末尾一致 a-a-i → 100% ✓
- ペアE「コーヒー/ストーリー」：`rhyme.py コーヒー ストーリー` → 80% ✓（o-o-i-i が共通）

全ペア67%以上。合格。

**フロウチェック：**
- 2行ずつペアで韻を踏む構成（AABB型）で一貫。フロウが安定する。
- 4小節ごとにテーマが切り替わり、起承転結が明確。
- 12行目「限界」と16行目「ストーリー」にパンチラインを配置。盛り上がりが2段階。

**ダブルミーニング・比喩チェック：**
- 8行目「俺には非常」→「非常に大事」の省略形。かつ「日常」の反対語「非常」としても機能。二重掛け。
- 12行目「眠気も疲れも超える限界」→「限界を超える」の慣用句を逆転させ、「限界」自体が超える主体のように見える構造。
- 15行目「相棒コーヒー」→ コーヒーの擬人化。パートナーとしてのコーヒー。

合格。

## 例2：「月曜日」（8小節）

<thinking>
ステップ1：お題の解釈
「月曜日」にまつわる情景・感情を洗い出す：
- 休日明けの憂鬱、だるさ
- アラームとの戦い
- 満員電車、通勤ラッシュ
- 布団から出たくない
- 「あと5分」の攻防
- 会社・学校への足取りの重さ
- 金曜日が遠い絶望感
- でも結局行く（サラリーマンの悲哀）

ステップ2：コンセプト決定
月曜日の憂鬱をユーモア寄りに描く。深刻になりすぎず、「みんなそうだよな」という共感を笑いに変えるトーン。パンチラインで「結局月曜に負ける俺ら」のオチに着地させる。

構成（8小節なので起承で構成）：
- 起（1-4小節）：月曜朝の地獄描写（アラーム、布団）
- 承（5-8小節）：通勤〜会社到着、オチ

ステップ3：キーワードの音分解
`uv run scripts/rhyme.py 月曜日` を実行：
```
月曜日 → げつようび (5拍)
  母音: e-u-o-u-i  子音: g-ts-y-·-b
```

関連キーワード：
```
単語      読み        拍  母音     子音
憂鬱      ゆううつ     4  u-u-u-u  y-·-·-ts
布団      ふとん       3  u-o-·    f-t-N
不満      ふまん       3  u-a-·    f-m-N

限界      げんかい     4  e-·-a-i  g-N-k-·

眠い      ねむい       3  e-u-i    n-m-·
だるい    だるい       3  a-u-i    d-r-·
```

ステップ4：韻ペア探索

**ルートA（LLM連想）：**
- 「布団」(u-o-N) → 「不満」(u-a-N)：50%だが子音 f 共通、「ふとん/ふまん」の音感が近い。布団の中で不満を言う情景。
- 「憂鬱」(u-u-u-u) → 「不屈」(u-u-u)：75%。憂鬱だけど不屈…のはずがない、という笑いに使える。
- 「限界」(e-a-i) → 「全開」(e-a-i)：100%完全一致。子音 g-N-k / z-N-k もほぼ同じ。月曜日に限界、でも全開で行くしかない。
- 「眠い」(e-u-i) → 「だるい」(a-u-i)：67%。u-i が共通。月曜朝の二大感情。

**ルートB（韻辞書検索）：**
`uv run scripts/rhyme.py --search 月曜日`
```
完全一致 (e-u-o-u-i): なし
類似パターン (類似度50%以上):
  80% e-u-o-u: 鉄道, 絶望, 月曜, 結合, ...
  80% e-i-o-u-i: 形容詞, 栄養士, 星条旗, ...
```

「絶望」(e-u-o-u) が80%一致で面白い。月曜日と絶望はテーマ的にも合う。

**マージして韻ペアを確定：**
- ペアA：「眠い」/「だるい」 → e-u-i / a-u-i、67%。起パートの状態描写。
- ペアB：「布団」/「不満」 → u-o-N / u-a-N、50%だが子音 f-N 共通で音感が近い。
- ペアC：「限界」/「全開」 → e-a-i / e-a-i、100%完全一致。★パンチライン候補。
- ペアD：「憂鬱」/「不屈」 → u-u-u-u / u-u-u、75%。★パンチライン候補。オチに最適。

ステップ5：リリック構築

各行を肉付けする：

1. アラーム五回 まだまだ眠い → 「眠い」(e-u-i)
2. 月曜の朝は 体中だるい → 「だるい」(a-u-i) ※67%
3. あと五分だけと 潜る布団 → 「布団」(u-o-N)
4. 目覚めた瞬間 出るのは不満 → 「不満」(u-a-N) ※50%+子音共通
5. 満員電車で すでに限界 → 「限界」(e-a-i)
6. 閉まるドアごと テンション全開 → 「全開」(e-a-i) ※100% ★パンチライン（皮肉）
7. 一週間の頭が 一番憂鬱 → 「憂鬱」(u-u-u-u)
8. けど月曜日だけは 心が不屈 → 「不屈」(u-u-u) ※75% ★パンチライン

ステップ6：検証

**韻の照合：**
- ペアA「眠い/だるい」：`rhyme.py 眠い だるい` → 67% ✓（u-i 共通）
- ペアB「布団/不満」：`rhyme.py 布団 不満` → 50% ✓（子音 f-N 構造が同一で音感を担保）
- ペアC「限界/全開」：`rhyme.py 限界 全開` → 100% ✓（e-a-i 完全一致、子音 N-k も共通）
- ペアD「憂鬱/不屈」：`rhyme.py 憂鬱 不屈` → 75% ✓（u-u-u が共通）

全ペア50%以上。合格。

**ダブルミーニング・比喩チェック：**
- 6行目「テンション全開」→ 皮肉。限界なのに全開という矛盾がユーモア。また「ドアが閉まる」と「テンションが開く」の開閉の対比。
- 8行目「心が不屈」→ 自分に言い聞かせてるだけの強がり。「憂鬱」との対比で「u-u-u」の韻が効いている。

合格。
</thinking>
